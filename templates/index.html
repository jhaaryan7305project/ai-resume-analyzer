About

AI Resume Analyzer is a smart web application that compares your resume against a real job requirements database and uses Gemini AI for detailed, company-specific feedback.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • AI Model: Google Gemini API
  • Job Database: Real requirements — 20 companies, 175+ roles
  • File Parsing: PyPDF2 (PDF), python-docx (DOCX)
  • Report Generation: ReportLab PDF Export

How the Score Works

  • 50% — Must-have skills match (from the job database)
  • 25% — Experience relevance & depth
  • 25% — Resume quality (clarity, formatting, impact)
